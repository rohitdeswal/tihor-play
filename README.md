# 🎮 tihor.play

A cozy collection of browser games built using pure **HTML, CSS, and JavaScript**.

This project is a fun little playground of classic games and interactive experiences — designed with clean UI, smooth interactions, and a playful aesthetic.

---

## ✨ Games Included

### 🎵 Simon Game
Test your memory by repeating an increasingly difficult color sequence.

### ❌⭕ Tic Tac Toe
Classic 2-player Tic Tac Toe with win detection and reset functionality.

### ✊📄✂️ Rock Paper Scissors
Challenge the computer in the timeless game of strategy and luck.

### ⚡ Who's That Pokémon?
Guess the Pokémon from its silhouette before time runs out.

### 🎩 The Sorting Hat
Answer magical questions and discover your Hogwarts house.
